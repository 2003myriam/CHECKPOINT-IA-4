# 🎬 CinéScope - Explorateur de Films

![CinéScope Banner](https://via.placeholder.com/1200x400/111111/e50914?text=CinéScope+%7C+Explorez+les+Films)

Une application web moderne et immersive pour explorer une collection de films cultes. Design cinématique, interface responsive et fonctionnalités interactives.

## ✨ Fonctionnalités

- 🔍 **Recherche en temps réel** - Filtrage instantané des films par titre ou catégorie
- 🌓 **Mode sombre/clair** - Thème dynamique avec persistance locale
- ❤️ **Système de favoris** - Sauvegarde locale des films préférés
- ⭐ **Notation visuelle** - Système d'étoiles basé sur les notes /10
- 🎨 **Effets modernes** - Animations, hover effects, glow et overlay
- 📱 **Design responsive** - Mobile-first, adaptation à tous les écrans
- 🎬 **Modal détaillée** - Visualisation des informations complètes des films
- 🖱️ **Scroll smooth** - Navigation fluide vers les sections

 

## 📋 Prérequis

- Navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Connexion internet (pour charger les images TMDB)
